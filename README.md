    [기능설명]

            Excel파일(.xlsx) Json 파일 변환 도구 

            xlsx -> (json, cs) 파일 변환 지원
        

    [사용방법]

          1. Drag and Drop을 통해 Excel(.xlsx)파일을 DragArea에 넣는다.

          2. 'Save JSON As' 버튼을 통해 json파일 저장 경로 지정 후, 확인한다.


          (cs 파일은 Scripts-Data 폴더에 자동 저장된다.)
